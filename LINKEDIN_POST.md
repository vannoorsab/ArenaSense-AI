Managing large-scale stadium events is always a challenge. Sudden crowd surges, congested exit gates, and delayed coordination can impact both user experience and safety.

To address this, I built **ArenaSense AI**—an intelligent stadium experience system that transforms raw data into actionable safety intelligence through real-time AI and high-fidelity simulations.

The system has been optimized for **production-level quality**, focusing on performance, security, and accessibility:

🔹 **Real-Time Crowd Intelligence**: Powered by Google Cloud Vision AI patterns to monitor 10+ stadium zones simultaneously.

🔹 **🛡️ Emergency Intelligence (New)**: One-touch evacuation mode with Panic Detection logic that automatically highlights danger zones and suggests the safest routes.

🔹 **🧪 System Validation Suite**: A built-in diagnostic dashboard that runs automated tests on AI and Cloud modules with real-time PASS/FAIL logging.

🔹 **Smart Gate Management**: Real-time entry/exit monitoring with AI-based load balancing to redirect fans to less crowded gates.

🔹 **Live Heatmaps**: Dynamic 2D/3D visualizations of stadium density, optimized with React memoization for 0ms visual lag.

🔹 **Accessibility First ♿**: Full WCAG 2.1 AA compliance with screen-reader optimizations and ARIA-labeled controls.

🔹 **Cloud-Powered Architecture**: Deep integration with Google Cloud Run, Cloud Logging, and scalable telemetry services.

Specifically for my fellow **Chennai Super Kings** fans 💛, I've integrated IPL 2026 matches where each match dynamically loads custom stadium layouts (Chepauk, Wankhede, etc.) and specific crowd simulations. This is stadium intelligence built for the "Yellow Army"!

This project is built as a mission-critical system designed to improve crowd flow, reduce waiting times, and most importantly, ensure everyone gets home safely.
